# CLAUDE.md — W3Sourcing (Frontend Clone)

## Project Overview
A modern recruitment/consulting website for W3Sourcing — a global recruitment firm specializing in Tech, Legal & Finance. This is a **frontend-only** Vercel project. No backend, no database. Static site with beautiful design.

## Brand: W3Sourcing
- Global recruitment for Tech, Legal & Finance
- Offices: London, Singapore
- Target: VC-backed tech companies, top law firms, global financial institutions
- Tone: Professional, premium, trustworthy, global

## Tech Stack
- **Framework**: Next.js 16 (App Router, TypeScript)
- **Styling**: Tailwind CSS + shadcn/ui
- **Font**: Inter or Geist (via next/font)
- **Package Manager**: bun
- **Deployment**: Vercel
- **No backend** — pure frontend/static

## Design Requirements
- Clean, premium, corporate feel (like goperfect.com but better)
- Hero section with gradient/glassmorphism
- Smooth scroll animations (intersection observer)
- Three service pillars: Tech, Legal, Finance — each with its own section
- "Why W3 Sourcing" section with 4 key differentiators
- "How We Work" section with 4-step process
- Industries & Functions grid
- Contact section with offices (London, Singapore)
- Mobile responsive
- Dark/light mode
- Professional footer

## Pages
- `/` — Main landing page (single page, scroll sections)
- `/contact` — Contact form (frontend only, no submission)

## Commands
- `bun dev` — development server
- `bun run build` — production build
- `bun run lint` — linting
